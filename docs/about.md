# Featured Projects

## AI LaTeX Repair Engine

One of my favorite projects is the **AI LaTeX Repair Engine**, an offline-first platform designed to simplify debugging AI-generated LaTeX documents.

Many AI-generated LaTeX files contain compilation errors that are difficult to understand, especially for beginners. Instead of requiring users to upload their documents to cloud services, I built a solution that performs PDF compilation directly inside the browser.

### Key Features

* Offline-first architecture
* Browser-based PDF compilation
* Monaco Editor integration
* Live PDF preview
* Human-readable compiler errors
* AI-ready repair prompt generation
* Minimal failing snippet extraction
* ZIP export workflow
* Unicode sanitization warnings
* Infinite loop protection
* Unsupported package detection

This project strengthened my understanding of browser-based computation, Progressive Web Apps, and building privacy-focused developer tools.

---

## Match Analyzer

Another project I'm proud of is **Match Analyzer**, a browser-based chess analysis platform powered by **Stockfish 18** running through WebAssembly.

Instead of sending chess games to a server, every analysis happens locally inside the browser.

The platform supports importing PGN files or Chess.com game URLs and generates detailed reports about player accuracy and move quality.

### Features

* Client-side chess analysis
* Stockfish 18 WebAssembly integration
* PGN import
* Chess.com URL import
* Accuracy reports
* Move classification
* Best move suggestions
* Evaluation graphs
* Completely free
* No accounts required
* No user data collection

This project introduced me to performance optimization, WebAssembly integration, and browser-based AI computation.
